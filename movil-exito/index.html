Una Telefonía para usarla

Icono

Móvil Éxito es el operador de telefonía del Grupo Éxito, el cual funciona con tecnología 4G LTE y la señal operada por Tigo.

La recarga de minutos y datos es prepago para que cada usuario decida cuánto quiere gastar y cuándo recargar. Además, hay cientos de puntos autorizados, tiendas de barrio, cabinas telefónicas, droguerías y todos los almacenes del Grupo Éxito para recargar Móvil Éxito.
